# 🌐 HeadySystems.com — Production Website

> **Live at [headysystems.com](https://headysystems.com)**

Production website for HeadySystems — the infrastructure backbone of the Heady AI ecosystem. Features dynamic content rendering, HeadyBuddy AI companion widget, Sacred Geometry design system, and multi-provider authentication.

## Features

- **Dynamic Multi-Domain Rendering**: Single server serves all Heady domain sites
- **Sacred Geometry Design**: Dark mode, glassmorphism, phi-scaled typography
- **HeadyBuddy Widget**: Embedded AI companion on every page
- **SEO Optimized**: JSON-LD, Open Graph, Twitter Cards, canonical URLs
- **WCAG AA Accessible**: Skip navigation, ARIA labels, 4.5:1 contrast ratios
- **Sentry Monitoring**: Real-time error tracking and performance monitoring

## Tech Stack

- Node.js 20+ on Google Cloud Run
- Cloudflare CDN/DNS
- Firebase Auth (25+ providers)
- Sentry error monitoring
